# Image 2.0 Company Pin Overlays

Transparent PNG marker assets generated with Image 2.0 for overlaying on a real map layer.

Use the `*-overlay.png` files for the app:

- `/map-overlays/company-pins/image2/pin-yellow-overlay.png`
- `/map-overlays/company-pins/image2/pin-blue-overlay.png`
- `/map-overlays/company-pins/image2/pin-mint-overlay.png`
- `/map-overlays/company-pins/image2/pin-red-overlay.png`
- `/map-overlays/company-pins/image2/pin-plum-overlay.png`

Suggested display size:

- Desktop map pins: `72px` to `96px`
- Mobile map pins: `56px` to `76px`

Suggested anchor:

- Use the bottom point of the pin as the geographic anchor.
- For the `512 x 512` overlay files, the anchor is roughly `x: 256px`, `y: 346px`.

The larger `pin-*.png` files are the full-resolution transparent exports. The `source/` folder contains the original chroma-key Image 2.0 outputs.
